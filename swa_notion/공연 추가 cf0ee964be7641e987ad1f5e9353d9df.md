API URL: /api/neticket/events

메인페이지에 관리자만
공연추가 버튼 활성화
Request: body
{
image: 이미지.jpg,
dto: {
"title": "공연명",
"place": "공연장소",
"price": 120000,
"date": "공연날짜",

}
